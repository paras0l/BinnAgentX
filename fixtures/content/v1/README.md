# 技术 Spike 固定内容包 v1

这里的六个内容单元均为项目自写的合成开发样例，只用于契约测试、状态流转和桌面 E2E。它们不是考研真题、外刊转载、教师金标或已完成难度标定的正式材料。

## 使用规则

- 先用 `contracts/content/v1/` 的 Schema 校验，再进入开发环境；
- 阅读正文哈希采用“按段落顺序、两个换行连接”的口径；
- 证据偏移以 JavaScript/Python 对 ASCII 正文的零基字符位置计算，`end` 为开区间；
- 任一文件修改都必须产生新的 `content_version_id` 和哈希，不得覆盖旧证据；
- 只有 Manifest 列出的版本可以被固定场景引用。

人工教研、目标用户难度试测与正式商用权利审核尚未完成。
