# 环境变量与运行边界契约

> 状态：工程骨架 v0.1
> 日期：2026-07-16

## 1. 分级

| 变量组                             |               是否秘密 | 浏览器可见 | 默认失败行为                     |
| ---------------------------------- | ---------------------: | ---------: | -------------------------------- |
| `BINNAGENT_ENV`、日志级别、时区    |                     否 |         否 | 非法值拒绝启动                   |
| `BINNAGENT_DATABASE_URL`           |                     是 |         否 | API 未就绪；不得回显连接串       |
| 身份适配器和控制角色               | 配置本身否；身份凭据是 |         否 | 生产环境使用合成适配器时拒绝启动 |
| 模型适配器、超时和预算             | 适配器名否；模型凭据是 |         否 | 预算或调用失败进入确定性降级     |
| 内容 Manifest 和最低权利状态       |                     否 |         否 | 权利不足时阻断呈现               |
| IRT/CAT、自动总分、预计提分开关    |                     否 |         否 | 默认关闭；环境缺失也保持关闭     |
| `NEXT_PUBLIC_LEARNER_API_BASE_URL` |                     否 |   仅学习端 | 缺失时相关网络功能不可用         |
| `NEXT_PUBLIC_CONTROL_API_BASE_URL` |                     否 |   仅控制舱 | 缺失时相关网络功能不可用         |
| OpenTelemetry endpoint             |       通常否；认证头是 |         否 | 本地允许不导出，不影响业务事实   |

## 2. 浏览器白名单

只有以下变量允许进入浏览器构建：

- `NEXT_PUBLIC_LEARNER_API_BASE_URL`；
- `NEXT_PUBLIC_CONTROL_API_BASE_URL`。

数据库、模型、认证、对象存储、控制角色、内容路径和功能开关都必须留在服务端。两个公开 API 地址也不能交叉注入：学习端构建只消费 learner 地址，控制舱只消费 control 地址。

## 3. 生产阻断

当 `BINNAGENT_ENV=production` 时，以下任一情况必须拒绝启动：

- learner 或 control 身份适配器为 `synthetic`；
- 模型适配器为 `deterministic_fixture`；
- `BINNAGENT_ENABLE_REMOTE_MODEL_CALLS` 未显式开启；
- 最低内容权利状态低于 `eligible_release`。

生产认证、模型供应商、部署地区和秘密管理尚未选择；因此当前代码和配置不构成生产部署许可。
