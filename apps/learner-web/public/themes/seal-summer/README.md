# 海豹夏日乐园运行时资产

本目录是 `docs/assert01/` 设计源素材的 Web 运行时副本。源文件保留原始中文文件名和完整分辨率；运行时副本统一使用 ASCII 文件名，避免路由、缓存和构建工具对路径编码的差异。

运行时品牌三主色固定为海洋蓝 `#319EE8`、珊瑚粉 `#FF8FA7`、阳光黄 `#FFD166`。深蓝、深粉等派生色只用于文字对比度和状态边界，不得取代三主色。

`ambient.jpg` 的生成目标：延续源素材的淡蓝水彩纸张质感，只在边缘安排气泡、贝壳、海星和柔和波纹，中央保持安静、无文字、无角色、无标志，以便承载真实界面内容。它不是新的品牌主视觉，也不得替代 `hero.jpg`。

独立角色使用统一的透明 PNG、完整轮廓和语义资产槽，不允许再从角色图集或 UI 总表中裁切界面素材。图集只承担图鉴和设计追溯职责。更新独立角色时必须检查透明角、半透明边缘、2× 显示清晰度、页面遮挡和按需加载。

典藏交互启用 `particle-headings` 与 `pointer-trail`：Pretext 成功建立标题场景后，页面 `<h1>` 保留原生语义和可访问名称，但视觉字形隐藏，由自适应字号的粒子画布承担可见标题；鼠标进入标题画布后，100px 半径内的粒子会让开并回弹。粒子脚本未建立场景、减少动态、打印或强制色模式下，原生标题会自动恢复可见；文章和表达任务的英文材料名使用 `<h2>`，不进入粒子系统。页面记录鼠标或触摸当前位置，只在滚轮、页面滚动或内层面板滚动时于该位置生成海蓝色气泡。气泡以单颗为主、快速滚动时偶尔成对，至少间隔 85ms，并使用半透明膜、高光弧、轻微形变和始终向上的漂浮轨迹；每颗气泡在出生时独立抽取，30% 会在生命末尾炸成一圈水环和六颗小水珠。

所有独立角色均通过内置图像生成，以三张源素材作为风格和角色参考，先在纯绿背景上生成，再使用本地柔边、去绿溢色流程转为透明 PNG。生成提示分别限定了页面语义、角色姿态、完整轮廓、无文字、无白底、无投影以及不得生成图集；原始生成文件保留在 Codex 的 `generated_images` 目录，项目仅引用裁边和压缩后的透明成品。
